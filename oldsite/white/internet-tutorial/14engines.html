Single Seach Engines

These engines search their own in-house-compiled database. Each search engine has a different search syntax and default mode.

They all offer an online help service, which you can access by clicking on the Help button on the toolbar, or by clicking on the Help link. We recommend that you use the online Help service to learn about the features for each particular search engine. Familiarizing yourself with an engine's capabilities will make you a more efficient searcher.

Google, Yahoo! Search, MSN, and Ask.com are examples of popular single search engines.
