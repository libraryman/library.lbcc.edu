Searching the Internet

After completing this tutorial, you will be able to

  • explain the difference between the Internet and World Wide Web
  • analyze a URL (uniform resource locator)
  • use search engines and metasearch engines to find information on the Web
  • use subject directories to locate information on the Web
  • interpret and evaluate Web search engine results
