What is a URL?

Each website has its own address, known as a URL, which stands for Uniform Resource Locator. A URL can tell you a lot about a website. For example, let's parse the following URL:

http://lib.lbcc.edu

http:// This site is accessible by HyperText Transfer Protocol
lib “lib” is short for Library. A library is the server for this site.
.lbcc “lbcc” stands for Long Beach City College. LBCC is the host site on the server.
.edu “edu” is short for “education.” This site is sponsored by an educational institution. This part of the URL is known as the top-level domain.
