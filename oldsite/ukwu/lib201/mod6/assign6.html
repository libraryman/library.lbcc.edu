Assignment 6: Reference Resources

The purpose of this assignment is to give you hands-on practice using reference resources. You will learn about the content and arrangement of some of the most-used reference tools in the library. As a result, you will acquire the skills necessary to identify and evaluate the similarities and differences between reference works.

Instructions

Take time to become acquainted with each reference resource: read the "how-to-use" chapter and refer to the list of abbreviations and symbols to help you interpret the information contained within. Read the instruction for each question carefully and answer all pertinent questions on a separate sheet. Many reference sources form a multivolume set; always check the index volume first to find the correct volume in which to find the answer.

If one of the resources you need is not on the shelf where it belongs, and you cannot find it, just write "NOS" as your answer. NOS stands for "not on shelf."

  1. Reference work: Statistical Abstract of the United States. Ref HA 202 .S7 (Shelved behind LAC Library Reference Desk. Please ask the Librarian.)
    Check the index for a table of statistics on televisions—households with. Hint: The index refers you to the table number—not the page number.
    1. For the most recent year reported, how many households had televisions? Note that the data is reported in millions, which means that you will need to add six zeroes after the number reported.
    2. What is the percentage of households with televisions?
  2. Reference work: CQ's State Fact Finder. Ref HA 214 .C72 (Shelved behind LAC Library Reference Desk. Please ask the Librarian.)
    1. For the most recent year reported, how many prisoners did California have?
    2. Which state had the least number of prisoners?
  3. Reference work: Encyclopedia of Associations. Ref HS 17 .E5 (Shelved behind LAC Library Reference Desk. Please ask the Librarian.)
    In order to answer the questions below, you will first need to use the index volume of this encyclopedia. Look for the volume that says "Index" on its spine. Look under the kewords in Question a. below. The index will direct you to an entry number—not the page number.
    1. Name an association based in Bethesda, Maryland, which supports medical research on cystic fibrosis.
    2. What is the URL for its website?
  4. Reference work: The Statesman's Yearbook. Ref JA 51 .S7 (Shelved behind LAC Library Reference Desk. Please ask the Librarian.)
    1. When did Croatia become an independent sovereign state?
    2. What is the capital of Croatia?
  5. Reference work: Gale Directory of Publications. Ref Z 6951 .A97 (Shelved behind LAC Library Reference Desk. Please ask the Librarian.)
    1. Who is the publisher of the newspaper Long Beach Express? Hint: Look under "California—Long Beach—Long Beach Express."
    2. What ethnic community in Long Beach does this newspaper serve?
  6. Reference work: Current Biography. Ref Bio Ref CT 100 .C8 (Shelved in the LAC Library Reference Room, along the Biograhy wall.)
    Consult the volume that contains the cumulated index, covering 1940-1995. The index volume is narrower than the others. You want to find biographical information about Walt Disney. The index will direct you to one or more annual volumes where you will find a biography about this person.
    1. In which annual volumes can you find biographical information about Walt Disney?
    2. In what year did the first Mickey Mouse cartoon appear?
  7. Reference work: Contemporary Authors. Ref Bio Z 1224 .C58 (Shelved in the LAC Library Reference Room, along the Biography wall.)
    In order to answer this question you will need to use the volume with the cumulative index (in soft-bound cover). Look up the author Sherman Alexie. The index will direct you to the volume where you will find information about this author.
    1. What type of literary genres does this author write?
    2. What is the title of the novel that won the American Book Award in 1996?
  8. Reference work: The Oxford English Dictionary. Ref PE 1625 .O899 (Shelved in the LAC Library Reference Room.)
    Look up the word "potlatch".
    1. From which language is this word derived?
    2. What is the definition of potlatch?
    3. What year is the earliest recorded use of this word?
  9. Reference work: Encyclopedia of Sociology. Ref HM 425 .E5 (Shelved in the LAC Library Reference Room.)
    You want to learn about the Hawthorne effect. Consult the index, then go to the page number where you can find information on this topic.
    1. What are the three most common explanations for the Hawthorne effect?
  10. Reference work: Masterplots (Revised Second Edition. Ref PN 44 .M372 (Shelved in the LAC Library Reference Room.)
    You are looking for information about the play, Lysistrata.
    1. Who wrote Lysistrata?
    2. What plan did Lysistrata have for ending the Second Peloponnesian War?
  11. Reference work: The Dictionary of Art (Grove). Ref N 31 .D5 (Shelved in the LAC Library Reference Room.)
    1. The ancient site of Copan in Honduras is famous for its sculptures. What civilization is associated with this ancient site?
    2. Name the two explorers who introduced this site to the world.
  12. Reference work: Oxford Dictionary of Quotations. Ref. PN 6080 .O95 (Shelved in the LAC Library Reference Room.)
    1. To whom is the following well-known quotation attributed? "A thing is not necessarily true because a man dies for it?" Think of the appropriate term or terms that you need to use to locate this information, then check the index and you will be directed to the page and line number where you can find this quotation, as well as the person to whom it is attributed.
    2. Give the page and the title of the work where this quote appears.
