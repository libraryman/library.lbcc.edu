Assignment 7: Searching Web-based Databases

The objective of this assignment is to allow you to apply concepts learned in this module into concrete search tasks. You will practice search techniques for Web-based databases and experiment with database selection.

Instruction

  1. Periodicals in Databases
    Go to the Library website at http://lib.lbcc.edu, and click on the button, "Online Databases." You will see a list of databases the Library subscribes to. Click on ProQuest and log in using the user ID and password given to you at the beginning of the course. Get passwords to Library databases
    ProQuest includes many discipline-specific modules. For this assignment, we explore only the following modules:
    • Health Module
    • Social Science Module
    • Education Module
    • Business Module
    • Psychology Module
    • Arts Module
    To access modules, click on "Select multiple databases" to the right of the search box. Scroll down to "ProQuest Research Library" towards the bottom of the list, and you will see the different modules listed. To view the titles of periodicals covered in a particular module, click on "View titles." Briefly scan through the first few pages of the list of publications of each module to see the titles of the publications included in each one. Then, answer the following questions:
    1. Do some modules contain the same periodical?
    2. List one periodical that appears simultaneously in different modules.
    3. Do many of the periodicals provide the full text of articles?
    4. Based on your observation of the list of periodicals from these modules, discuss the importance (i.e., the why and how) of doing research in appropriate discipline-specific modules.
  2. Database Selection
    The first important step in research is to select the most appropriate database for the topic. Although the six topics listed below can be searched in one or two modules, I want you to select six different modules, each most closely matched to the topic, as the purpose of this assignment is to acquaint you with various discipline-specific modules. Discuss your search for the six topics listed below by answering the questions.
    1. Topic: computer graphics
      Which module did you select for this topic? Why?
      What type of periodicals are indexed in this database (i.e. popular or scholarly or both)?
      Give the exact search terms you used for this search.
      Give one relevant citation from you search result.
    2. Topic: corporate scandals

    4. Topic: cognitive behavior therapy for treating bulimia

    5. Topic: distance education

  3. Boolean (Advanced) Search
    1. Perform a Boolean search on the topic of "affirmative action and college admissions in California."
      List the search terms, the fields, and the Boolean operators you used for this search.
      List one relevant citation from you search result.
    2. Perform a Boolean search on the topic of "affirmative action and college admissions but not in California." List the search terms, the fields, and the Boolean operators you used for this search.
      List one relevant citation from you search result.
    3. Note the number of hits obtained for each search and discuss the difference.
