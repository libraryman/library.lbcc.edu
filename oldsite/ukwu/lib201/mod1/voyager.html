Searching Voyager

Simple Search

Voyager provides you with many search options. The image above represents the screen used to input a simple search. Notice the light grayed-out tab on the top left, which indicates that you are now in the Simple Search mode.

Select a search option from the drop-down menu on the right by pointing and clicking on the option. The option will be highlighted in blue. The example above shows that the Keyword search was selected. The search options are: Keyword search, Title search, Author search, Command Keyword search, Subject heading search, Periodicals search, Subject Word or Phrase search, Call number search.

A Simple Search scans an index and returns every subject, author, title, etc. that begins with the term(s) you type in the dialog box. For example, a search on the subject "medi" would return the topics: medicaid, medical, medici, medicine, medieval and mediterranean. You can search by subject, author, title, periodical title, call number, and so on.

Searching Hints

  • Enter the subject, title, or author name, omitting all punctuation
  • Truncation (a stem search) on the right is implied
  • When searching by author, enter the last name first

Keyword Searching Hints

A Keyword search looks for words located anywhere in a record. For example, a Keyword search on the word "computer" would return all of the titles that contain the word computer anywhere in the record.

To perform a Keyword search:

  1. Type the word(s) you want to search for in the appropriate text box.
    Use a question mark (?) after a search term to indicate truncation: comp?
    Use quotation marks to search for specific phrases: "world wide web"
    Use a plus sign (+) before a term to indicate an essential term: +internet
    Use an asterisk (*) before a term to indicate an important term: *explorer
    For example: comp? "world wide web" +internet *explorer
  2. Click the Set Limits icon if you want to limit your keyword search based on other criteria.
  3. Click on the drop-down menu to select the number of records you want to display on a page.
  4. Click the Search button to begin your search. Click the Reset button to clear the search page.

Voyager Advanced-Search Mode

An Advanced search finds records using keywords located anywhere in a record, or only in specific fields. To perform an Advanced search:

  1. Type in the search term(s) you want to find in the dialog box next to the "Search for" option. (Punctuation, case, and word order are ignored.) Your search terms are automatically truncated at the root level of the word (e.g. comput, educat, institut)
  2. Default search criteria can be modified for your search:
    1. Select from the drop-down list whether you want to search for all the search terms, any of the search terms, or the search terms as a phrase.
    2. Select from the drop-down list next to the Search by option whether you want to search by title, author, subject, and other options.
    3. Select which Boolean operator you want applied to the next search term(s): AND, OR, or NOT to further limit your search.
  3. Click the "Set Limits" icon if you want to limit your search based on other criteria.

Click the "Search" button to begin your search. Click the "Reset" button to clear the search page.

Voyager Course-Reserve-Search Mode

A Course-Reserve search allows you to search for items on reserve for various classes. To perform a Course-Reserve search:

Select one item from one (or more) of the following drop-down lists: Instructor, Department, Course, or Section. Note: there is no "Set Limits" option for a course-reserve search.

Click the "Search" button to begin your search. Click the "Reset" button to clear the search page.

Voyager View Your Record Search

You must log in in order to access your patron information, such as the status of any pending requests, items currently on loan to you, and any fines or fees owed by you.

Enter your social security number and your last name in the Last Name dialog box and your college I.D. number in the College ID# dialog box.

After you have entered your Social Security # (no dash between the numbers) and last name, click on "Login". Click on "Reset" to clear the dialog boxes.
