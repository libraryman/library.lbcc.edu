Module 5 - Print Indexes - Graded assignment


The purpose of this assignment is to acquire skills in interpreting citations, identifying main subject headings in print periodical indexes, and in selecting appropriate indexes for specific topics. Although some topics are covered in different indexes, I want you to select the index that is most closely matched to the topic in question.

Instruction


You are asked to find periodical articles on five topics, written between 1997-1999. You will use only H.W. Wilson print periodical indexes; remember that these indexes are published yearly in a bound volume with the time period year marked on the spine (e.g. Humanities Index yearly bound volume is from April the previous year to March the following year: April 1998-March 1999). There are also supplements published quarterly in soft-bound volumes. Search LBCC library catalog or any library catalog near your home for the call number and the location of the indexes. The five periodical indexes to be used for this assignment are:


1. Art Index ­ November 1997-October 1998
2. Business Periodicals Index ­ August 1997-July 1998
3. Education Index ­ July 1998 ­ June 1999
4. Humanities Index ­ April 1998- March 1999
5. Social Sciences Index ­ April 1998- March 1999


For this assignment, use the bound volumes listed above. Use one different index for each topic. Answer all questions related to each topic. List all citations in MLA style.
