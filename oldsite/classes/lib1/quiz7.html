Module 7 - Electronic databases - Graded assignment

The objective of this assignment is to allow you to apply concepts learned in this module into concrete search tasks. You will practice search techniques for Web-based databases, and experiment with database selection.

Instruction

Go to the library Web site at http://lib.lbcc.cc.ca.us, you will see a list of database names under the Databases heading. Select ProQuest Direct by clicking on the name link and log-in using the user ID and password given you at the beginning of the course. The next screen lists all the subscribed databases; you can see that these are discipline-specific databases. Explore one by one the following databases:
