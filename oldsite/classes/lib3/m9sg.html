Introduction

The Internet is a network of computers worldwide linked together to exchange information using communication softwares and phone lines. The Internet is a very volatile environment in the sense that new features and capabilities are constantly updated. New sites and new files are created daily and old files disappear almost at the same rate.

Web sites and Web pages
These terms are used interchangeably. However, to be more precise, a Web page is one single page of a Web document and can be as long or as short as the author wants it to be. A Web site is comprised of many Web pages or a single Web page.

Internet protocols
The exchange of information over the Internet is performed using various protocols. They are: e-mail, ftp (file transfer protocol), telnet, http (hypertext transfer protocol), gopher, etc. The most popular protocols are e-mail and hypertext transfer protocol allowing information exchange via web-based files and thus forming the World Wide Web.

Free and Fee-based Internet
Most information on the Internet is free from Web sites sponsored by governmental, educational, non-profit institutions, and big corporations selling their products and services. Other information is fee-based and reserved only for paying subscribers with users ID and passwords. One example of fee-based service is ProQuest Direct, a commercial database accessible via the Internet but reserved only to paying subscribers such as LBCC Library and its campus users.

Browsers

Uniform Resource Locator (URL)
Each Web site has its own Internet address, more properly called Uniform Resource Locator or URL. A URL is composed of the following sequential elements:
(1) the protocol, (2) the name of the host computer, (3) the domain or organization's name, (4) the directory name, and (4) the file name. For example:


http://www.governor.ca.gov/gray/index.shtm

A URL usually starts, but not always, with www.
The top level domain indicates the nature of the Web site:

.edu denotes a Web site sponsored by an educational institution (e.g. ucla.edu).
.gov denotes a Web site sponsored by a governmental institution (e.g. state.gov).
.mil denotes a Web site sponsored by a military institution (e.g. navy.mil).
.com denotes a Web site sponsored by commercial institution (e.g. disney.com).
.org denotes a Web site sponsored by an organization (e.g. apa.org).
.net denotes a Web site sponsored by an Internet group (e.g. alaska.net).

The URL can tell you much about a web site. For example:

http://lib.lbcc.cc.ca.us/kbarclay/index.html

Subject Directories
A subject directory is a database of Web-based files organized hierarchically by subject matter. Imagine a hierarchy tree beginning on the top with a broad subject or main heading, then organized further into subdivisions of the broad subject. Subject directories are good for browsing when you do not have a specific topic in mind or when you are not sure of the terms or the names to use for your search.

There are two types of subject directories:

  • Commercial subject directories (e.g. Yahoo, etc.) have no or very few annotations and are not evaluated.
  • Academic subject directories (e.g. Infomine, Librarian's Index to the Internet) are reviewed by librarians and information specialists; detailed annotations are given with each listed Web sites.

Search Engines
A search engine is a database of Web-based files compiled by a software program. A search engine has a built-in software to read and respond to users' queries. Search engines are useful when you have a specific topic in mind, or when you know the terms or the names to use for your search.

There are two types of search engines:


1. Single search engines (e.g. Altavista, Lycos, HotBot, etc.) search only their own in-house compiled database. Each search engine has different search syntax and default mode. They all offer online help service; look for the Help button or Help link to learn more about each search engine's capabilities. I recommend that you use the online Help of each search engine to get the most of what each has to offer. Search engines may overlap each other with regard to Web sites they include in their database; however, they also differ greatly from each other. It is a good idea to familiarize yourself with various search engines; sometimes, you may have to use a few search engines to find what you need.


2. Meta search engines (e.g. Metacrawler, etc.) searches different search engines at the same time.

To learn more about subject directories and search engines, visit: http://www.albany.edu/library/internet/

 

Evaluation of Internet Resources
Internet resources are subject to the same evaluation criteria as for printed resources:

  • Currency
  • Author
  • Publisher
  • Depth
  • Bias


It is more difficult to evaluate Internet resources because often the author's name is not so obvious to see. Some Web sites do not give out information such as publication date or when the Web site was last updated. When you visit a Web site, look for the update date generally found at the bottom of the page. The sponsoring institution of the Web site would give a good indication about the credibility or authority of the site. You need to determine who is the target audience for the site in question. Measure the depth of information given out from this site. Look for bias: does the site aims to inform, or to educate, or to proselytize, or to publish its propaganda? Check the links included in this site; too many dead links would indicate that this is a static Web site and the information it contains may not be current.

More information about evaluating Internet resources can be found at: http://www.albany.edu/library/internet/evaluate.html

Citation of electronic resources

The two main styles of citation for electronic documents are the same as for printed materials: the MLA style and the APA style. Refer to the Web sites of both associations listed below for detailed instructions on how to cite electronic resources.

MLA style: http://www.mla.org/style/sources.htm

Capturing electronic documents


There are four ways to capture electronic documents:


1. Print ­ To print an electronic document, click on the Print button on the tool bar, or go to File, then select Print.

2. Save ­ To save an electronic document to a disk, insert a disk into the disk drive of your computer, then go to File, select Save As and follow the prompts in the dialog box. Remember to select the proper disk drive and to name your document.

3. E-mail ­ for Netscape users, to e-mail an electronic document, go to File, select Send Page, a dialog box will prompt you to perform certain tasks, follow the prompts in the dialog box. Instructions may vary with a different Web browser or with different versions of Netscape. Many commercial databases such Proquest Direct and NewsBank have built-in e-mail capacity, you need only to click on the e-mail button, type in the e-mail address, then click on Send or Submit.

4. Copy and Paste ­ To copy and paste a whole or a portion of an electronic document into a word processing application, first highlight the part of the electronic document you want to copy, go to Edit, select Copy, then open a word processing program on your computer such as Microsoft Word or Simple Text, create a new file and go to Edit, select Paste.
