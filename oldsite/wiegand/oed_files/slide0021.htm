What is a Wildcard?
nA wildcard is a character which stands for any letter.  You can include wildcards
nIn a Find Word or Full Text search to make your search more flexible.  There are
ntwo wildcards available in OED:
n
nThe question mark ? Represents the occurrence of any one single character.
n For example:   c?t finds cat, cot, cut
n
nThe asterisk * represents the occurrence of any number of characters (or no character at all).
n For example:  c*t finds cat, caught, commencement, conflict, consent, cot, cut, etc.
